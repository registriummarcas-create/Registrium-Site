# Tally

Campos recomendados:

- Nome — obrigatório
- WhatsApp — obrigatório
- E-mail — opcional
- Nome da marca — obrigatório
- Segmento — obrigatório
- Cidade e estado — opcionais
- Situação perante o INPI
- Observações
- Consentimento para contato — obrigatório

Campos ocultos:

- `utm_source`
- `utm_medium`
- `utm_campaign`
- `utm_content`
- `utm_term`
- `landing_variant`
- `page_url`
- `referrer`

Configure a conclusão para `/obrigado` e a notificação para o e-mail institucional da Registrium.
